# 带薪看片(paid-to-watch)

> 一个常驻桌面右下角、长得像流氓广告弹窗的本地视频播放器。真视频藏在「假广告」的注意力盲区里——上班摸鱼看片,旁人扫一眼只当是个垃圾弹窗广告。

**定位**:纯粹好玩的开源工具。唯一 KPI 是「让打工人会心一笑、点个 star、转给同事」。**不做任何变现**(无广告联盟、无恰饭位)。设计原则一条:**梗 > 功能**,能好玩绝不无聊。

---

## 核心机制:为什么这套伪装有效

不是「藏」视频,而是**反向利用注意力盲区**。右下角弹窗广告,人人本能屏蔽,越花哨越没人看第二眼。本工具把真视频明晃晃摆在那,靠「它看起来像垃圾广告」让旁人的目光自动滑过去。**藏不如露**——上下两条假广告不是装饰,是伪装色。

---

## 技术栈

- **Electron**(HTML / CSS / JS)。理由:假广告外壳用 HTML/CSS 随便造、想多骚有多骚;视频原生支持;跨平台 Mac/Win,开源传播面最广。
- 目标平台:**Windows 优先**,Mac 兼容。

---

## 功能(MVP)

### 1. 窗口
- 无边框小窗,默认贴屏幕右下角,**永远置顶**
- 三段式纵向布局:上横幅(假广告)/ 中间(视频)/ 下横幅(假广告)
- 可拖动、可调大小;关闭 / 设置 / 退出藏在**系统托盘**
- 记住上次的位置、大小、视频文件夹

### 2. 假广告外壳(伪装色)
- 预置数套牛皮癣模板:领红包、清理加速、附近的人、股票涨停、9.9 包邮
- 上下两条**定时轮播**,模拟真广告刷新;视觉越刺眼越 low 越真
- 角落一个假的「× 关闭」:**点了假装关掉、立刻又弹回来**(复刻流氓广告关不掉的烦人感,既是梗又强化伪装)

### 3. 视频区
- 启动时指定一个本地文件夹 → 扫描出视频列表 → 点选即播
- 画面纯净:平时不显控件,鼠标悬停才出极简进度条
- **默认静音**(外放出声是这类工具最大破绽,首版只做静音观看)

### 4. Boss Key(灵魂功能)
- 一个全局热键(默认 `Ctrl+Alt+Z`,设置中可改)
- 按下:**中间视频区无缝替换成第三条假广告**——不是最小化(最小化反而显眼),是「秒变满屏广告」,旁人凑近也只看到广告
- 再按一下:变回视频,从刚才的位置继续

## 开源 / 传播策略(传播是这个项目的命脉)

- **README 走自黑梗**:「史上最敬业的摸鱼工具 / 让老板以为你中了流氓软件」
- **一个 GIF demo**:老板走近 → 按键 → 视频秒变广告。这个 GIF 是传播引爆点,必须做好
- **MIT 协议**
- **提供打包好的 release**:让不会编译的打工人也能直接下载即用(对传播极关键)

---

## 首版不做(YAGNI)

在线视频(B站/YT)、字幕、耳机检测后出声、自定义 / 自换假广告图、换肤主题、任何变现钩子。

---

## 已知边界与风险

- **声音**:首版只支持静音观看,想听声音得自己戴耳机手动开(留到后续做耳机检测)。
- **调性**:工具帮人上班摸鱼,作为开源玩具没问题(GitHub 上同类 boss-key 工具很多),但 README 不主动教唆、不承诺「绝对不被发现」,附好免责声明。
- **假广告素材**:用原创 / 虚构的广告内容,不照搬真实品牌素材,避免商标问题。
