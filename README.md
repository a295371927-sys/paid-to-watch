# 带薪看片 (paid-to-watch)

> 史上最敬业的摸鱼工具——把视频藏进一个「假广告弹窗」里。老板路过,只当你又中了流氓广告。

![demo](docs/demo.gif)

## 它是什么

一个常驻桌面右下角、长得像垃圾广告弹窗的本地视频播放器。上下两条是**假广告**(伪装色),中间偷偷播你自己的视频。一个热键就能把视频秒变全屏广告。

## 用法

```bash
npm install
npm start
```

- 右键**托盘图标** → 选择视频文件夹(支持 `.mp4 .webm .ogv .m4v`)
- 鼠标移到视频区底部 → 播放列表
- **Boss Key:`Ctrl+Alt+Q`** → 视频 ↔ 全屏假广告 一键切换
- 那个「×」是假的,点了它会「关不掉」😈
