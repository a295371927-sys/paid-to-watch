# 带薪看片 (paid-to-watch)

> 史上最敬业的摸鱼工具——把视频藏进一个「假广告弹窗」里。老板路过,只当你又中了流氓广告。

## 它是什么

一个常驻桌面右下角、长得像垃圾广告弹窗的本地视频播放器。上下两条是**假广告**(伪装色),中间偷偷播你自己的视频。一个热键就能把视频秒变全屏广告。

## 用法

```bash
npm install
npm start
```

- 右键**托盘图标** → 选择视频文件夹(支持 `.mp4 .webm .ogv .m4v`)
- 视频播完自动切到全屏假广告;**Boss Key:`Ctrl+Alt+Q`** → 视频 ↔ 全屏假广告 一键切换,文件夹有多个视频时循环播放下一个
- 右键托盘图标 →「设置老板键」可自定义快捷键组合
- 那个「×」是假的,点了它会「关不掉」😈

## 网页直播

视频区也能叠加显示网页直播,看起来更像广告里嵌的「推荐视频」:

- 右键托盘图标 →「网页直播」开关,叠加显示直播网页;「直播声音」单独控制是否静音
- 「收藏网站」子菜单快速切换直播源(默认含 CCTV5 / 咪咕视频 / B 站直播)
- 「直播设置模式」临时放大窗口并显示地址栏,可输入任意网址跳转,关闭后自动恢复原伪装尺寸
- 按 Boss Key 期间网页直播会自动隐藏并静音
